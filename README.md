## Installation

- Install vagrant

- Import Ubuntu 12.04 box

  ```
  vagrant box add ubuntu-12.04 https://s3-us-west-2.amazonaws.com/squishy.vagrant-boxes/precise64_squishy_2013-02-09.box
  ```

- Start the vm

  ```
  vagrant up
  ```

export ROM_ROOT="/path/to/clones/of/rom/projects"
